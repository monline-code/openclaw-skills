- **版本**: 1.0.0

- **位置**: `~/.openclaw/workspace/skills/stock-realtime-fetcher/`

## 🎯 核心功能

提供 **A 股实时行情数据获取** 功能，支持 **5 个数据源自动故障转移**：

1. **腾讯财经** ⭐ - 稳定快速，默认首选
2. **东方财富** - 数据全面，备选方案
3. **AkShare** - 历史 K 线，有延迟
4. **Baostock** - 免费免注册
5. **Tavily** - 搜索获取，需 API 密钥

## 🚀 使用方式

### 方式 1: 命令行

```bash
# 单只股票
python3 ~/.openclaw/workspace/skills/stock-realtime-fetcher/stock_fetcher.py 600519

# 多只股票
python3 stock_fetcher.py 600519 000001 300750

# JSON 格式
python3 stock_fetcher.py 600519 --json

# 指定数据源
python3 stock_fetcher.py 600519 --prefer eastmoney
```

### 方式 2: Python 调用

```python
import sys
sys.path.insert(0, '~/.openclaw/workspace/skills/stock-realtime-fetcher')
from stock_fetcher import get_stock_data, get_multiple_stocks_data

# 单只股票
data = get_stock_data("600519")
print(f"贵州茅台：¥{data['price']} ({data['change_pct']:+.2f}%)")

# 批量获取
stocks = get_multiple_stocks_data(["600519", "000001", "300750"])
```

### 方式 3: 集成到其他技能

```python
# 在其他技能的代码中
FETCHER_PATH = "~/.openclaw/workspace/skills/stock-realtime-fetcher"
import sys
sys.path.insert(0, FETCHER_PATH)
from stock_fetcher import get_stock_data

# 获取实时数据（自动故障转移）
data = get_stock_data(symbol)
```

## 🔧 配置说明

### 修改数据源优先级

编辑 `stock_fetcher.py` 第 23 行：
```python
DATA_SOURCES = ["eastmoney", "tencent", "akshare", "baostock", "tavily"]
```

### 调整超时时间

编辑 `stock_fetcher.py` 第 26 行：
```python
TIMEOUT = 10  # 默认 5 秒
```

### 配置 Tavily API（可选）

```bash
export TAVILY_API_KEY="tvly-dev-xxxxxxxx"
```

或创建配置文件 `~/.openclaw/workspace/tavily_config.json`

## 🆚 与 stock-data-fetcher 的区别

| 特性 | stock-data-fetcher | stock-realtime-fetcher |
|-----|-------------------|----------------------|
| 定位 | 数据获取模块 | 独立可复用技能 |
| 文档 | 简单说明 | 完整 SKILL.md |
| 示例 | 无 | 包含 examples.py |
| 元数据 | 无 | _meta.json |
| 快速指南 | 无 | QUICKSTART.md |
| 适用场景 | 内部调用 | 独立技能/被其他技能调用 |

**建议**: 
- 需要独立技能时使用 `stock-realtime-fetcher`
- 内部模块调用可使用 `stock-data-fetcher`
